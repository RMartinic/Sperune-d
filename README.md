# 🍽️ Šperun Restaurant Website

Static website built for **Šperun Restaurant** that presents restaurant information, menu, news, and gallery in a clean and responsive layout.

The website focuses on **mobile-friendly design** and provides a **multilingual menu system** powered by JavaScript and JSON.

---

## 🌐 Live Demo

🔗 [Vercel](https://sperune-d.vercel.app/)

---

## ✨ Features

- 📱 **Responsive mobile-first design**
- 🍽️ **Menu section with 5 language translations**
- 📰 **News section dynamically loaded from JSON**
- 🖼️ **Image gallery**
- 🌍 **Client-side language switching using JavaScript**
- ⚡ **Static site optimized for fast loading**

---

## 🛠️ Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Data Handling**
- JSON (for menu translations and news content)

**Deployment**
- Vercel

---
## ⚙️ How It Works

- Menu translations are stored in a **JSON file**
- JavaScript dynamically loads the correct language version
- News articles are also **loaded dynamically from JSON**
- The site updates content without needing backend changes

---

The website is deployed and can be accessed directly through the Vercel link:
